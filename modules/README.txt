Modules extend your site functionality beyond the functionality provided by the
core itself.

WHAT TO PLACE IN THIS DIRECTORY?
--------------------------------

Downloaded and custom modules are placed in this directory. This allows CMS core
light-weighted and independent updation of core and modules.

ORGANIZING MODULES IN THIS DIRECTORY
------------------------------------

You may create subdirectories in this directory, to organize your added modules,
without breaking the site.

NAMING AND PLACING STANDARDS
----------------------------

Modules are created and placed in this directory as per the following rules:
./<Module-Name>/

create a directory with your Module Name, and place anything you want to in that 
directory, you can have seperate robots.txt file for your module, if you want more 
enhanced security for your modules.
